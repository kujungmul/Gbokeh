# Gbokeh















1. Install R on ubuntu. 
  - [in terminal]
  - sudo apt-get install r-base-core

2. Install igraph on R
  - [in terminal]
  - R
  - install.packages("igraph")
  
3. Install rJava 
  -[in terminal]
  - sudo apt-get install r-cran-rjava
  - sudo apt-get install libgdal1-dev libproj-dev
  - R
  - install.packages("rJava")
